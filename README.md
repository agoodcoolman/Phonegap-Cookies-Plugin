Phonegap Cookies Plugin
=======
ionic 插件 
功能： 用来清除本地的cookie哒。 android/ios都可以哒。

```
$ phonegap local plugin add https://github.com/bez4pieci/Phonegap-Cookies-Plugin.git
```

## Usage

```javascript
window.cookies.clear(function() {
	console.log('Cookies cleared!');
});
```
